MultiSet

MultiSet 是 Set 的延伸, MultiSet 是继承 Collection 实现的, 它可以分成两种方式来使用, 将 MultiSet 看做一个普通的 Set 时, 是一个允许出现重复的元素 Set , 因此和 JDK 自带的 Set 不同, MultiSet 允许出现重复的元素, 当看成 Map<E, Integer> 类型时, 则表示键为元素, 值为计数的 Map 类型, 可以方便的对键的值进行计数
