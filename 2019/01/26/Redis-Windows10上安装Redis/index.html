Redis-Windows10上安装Redis

发表于 2019-01-26 | 分类于 Redis

Redis 天然支持 Linux 系统, 不支持 Windwos 系统, 因此微软团队在 GitHub 上开发了一个 Redis Win64 的版本, 下载地址

安装

下载后, 解压即安装, 解压后文件目录如下图所示

主要用到的文件如下

  • redis-cli.exe 客户端
  • redis-server.exe 服务器
  • redis.windows.conf 配置文件

运行 Redis

双击 redis-server.exe 就可以运行 Redis 了, 但是直接运行并没有使用配置文件, 我们可以在启动的时候来指定配置文件, WIM + R, 输入 cmd, 然后切换到 Reids 解压目录下, 执行如下指令:
